# convert_fsdp_checkpoint.py 使用教程

## 📖 简单三步走

### 第一步：找到你的 checkpoint 目录

你的 6 个 `.distcp` 文件应该在一个目录里，比如：

```
/path/to/checkpoints/
├── epoch_100/                    ← 这是你的 checkpoint 目录
│   ├── checkpoint_0.distcp
│   ├── checkpoint_1.distcp
│   ├── checkpoint_2.distcp
│   ├── checkpoint_3.distcp
│   ├── checkpoint_4.distcp
│   └── checkpoint_5.distcp
```

### 第二步：运行转换命令

```bash
python convert_fsdp_checkpoint.py /path/to/checkpoints/epoch_100
```

### 第三步：获得完整的 .pth 文件

转换完成后，你会得到：

```
/path/to/checkpoints/epoch_100.pth  ← 合并后的完整文件
```

## 🔍 如何找到你的 checkpoint 目录？

### 方法 1：如果你知道大概位置

```bash
# 在某个目录下搜索 .distcp 文件
find /path/to/your/workspace -name "*.distcp" | head -10
```

### 方法 2：搜索整个用户目录

```bash
# 搜索你的用户目录（可能需要几分钟）
find $HOME -name "*.distcp" 2>/dev/null
```

### 方法 3：查看最近修改的目录

```bash
# 查找最近 7 天内修改的包含 .distcp 的目录
find /home/user -name "*.distcp" -mtime -7 2>/dev/null
```

## ❓ 常见问题

### Q1: 提示找不到 checkpoint 目录

```bash
# 错误示例
python convert_fsdp_checkpoint.py /wrong/path
# 错误: Checkpoint 目录不存在: /wrong/path
```

**解决方法：**
```bash
# 1. 检查路径是否正确
ls /wrong/path

# 2. 使用 find 命令找到正确路径
find ~ -name "*.distcp" | head -1
# 输出: /home/user/experiments/dinov3/checkpoints/ckpt_epoch100/checkpoint_0.distcp

# 3. 使用正确的目录路径（去掉文件名）
python convert_fsdp_checkpoint.py /home/user/experiments/dinov3/checkpoints/ckpt_epoch100
```

---

### Q2: 提示没有找到 .distcp 文件

```bash
# 错误: 在 /path/to/dir 中没有找到 .distcp 文件
```

**可能原因：**
- 路径指向了单个文件而不是目录
- checkpoint 使用了其他格式（如 `.pt`, `.pth`）

**检查方法：**
```bash
# 查看目录内容
ls -la /path/to/dir/

# 如果是其他格式，可能不需要转换
# 直接使用 torch.load() 即可
```

---

### Q3: 转换后文件很大

这是正常的！分布式 checkpoint 分片后的总大小 = 合并后的大小

```bash
# 查看原始文件大小
du -sh /path/to/checkpoint_dir/
# 输出: 1.2G

# 查看转换后文件大小
du -sh /path/to/checkpoint_dir.pth
# 输出: 1.2G  (应该差不多)
```
